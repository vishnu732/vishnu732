## `> featured_project`

### Agentic Workflow Automation Engine

A full-stack AI workflow automation platform that converts plain-English business process descriptions into executable multi-step workflows.

```bash
Input   →  Customer refund request
Planner →  Gemini + LangChain task decomposition
Tools   →  Mock enterprise tool execution
Backend →  FastAPI REST APIs
Storage →  SQLite workflow history
Human   →  Approve / Reject / Retry
Metrics →  Task completion rate + workflow completion rate
```

**Core Features**

* Gemini + LangChain structured task decomposition
* FastAPI backend with REST APIs
* React workflow monitoring dashboard
* SQLite workflow persistence
* Mock enterprise tool calls
* Audit logs for every agent action
* Human approve/reject controls
* Retry workflow functionality
* Evaluation metrics and task completion tracking
