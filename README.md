## Featured Project

### Agentic Workflow Automation Engine

A full-stack AI workflow automation platform that converts natural language business process descriptions into executable multi-step workflows.

**Tech Stack:** Python · FastAPI · LangChain · Gemini API · React · SQLite · REST APIs

**Key Features:**

* Gemini + LangChain task decomposition
* FastAPI REST backend
* React workflow monitoring dashboard
* Mock enterprise tool execution
* SQLite workflow history
* Audit logs for every agent action
* Human approve/reject controls
* Retry workflow functionality
* Evaluation metrics including task completion rate
